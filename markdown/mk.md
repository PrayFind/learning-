### 一、准备工作
1.**安装vscode**
2.**下载必要插件**
 -Markdown All in one 
 -Markdown Preview Enhanced
 -Markdown PDF 
 -Paste Image
3.**创建.md文档，打开同步预览功能，开始编辑**
### 二、基本语法
1. **标题**
  # 一级标题
 ## 二级标题
 （#+空格）
2. **引用**
> 编辑这类教程文档很好用
>嵌套引用（>> sss)
>> ss

> 硕士

（引用结束需要额外空一行）
3 **列表**
 1.无序列表
   - 列表1
   - 564
   + 列表2
   + 列表3
   * ss
 2.有序列表
 3.嵌套（在列表以内加入tab）
 4.Todolist
   - [x] a 
   - [ ] b
   - [ ] c 
4.**表格**

| 左对齐 | 右对齐 | 居中对齐 |
| :---- | ----: | :---: |
| a | ba | v |


表格上下需要空行
表格生成失败，多复制粘贴尝试

| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 内容1 | 内容2 | 内容3 |
| 内容A | 内容B | 内容C |

5.段落 
   - 换行：连续两个以上空格后换行
   - 分割线：三个以上*
   - ***
   - 字体
   | 字体 | 代码 |
   | :---: | :---: |
   | *斜体* | * * |
   | ==高亮== |== ==|
   |**粗体**|** **|
   |***铅斜体***|*** ***|
   |~~删除~~|~~ ~~|
   -脚注（本质引用，前有，后面再加一个[^1]:注意有冒号
   请大家多多三[^1]


[^1]: 点赞投币收藏

6.代码展示
   - 单句展示
   `int = a`

   - 多局展示（注意要空行）
   ``` 
   ingghh = 1
   ssgg == 0
   tt
   ```
